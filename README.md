# 🧠 Project Overview

This project investigates digital inequality in Indonesia by analyzing disparities in internet access across provinces, urban–rural segmentation, and socioeconomic indicators such as the Human Development Index (HDI).

The objective is to uncover structural gaps in digital infrastructure and provide evidence-based insights to support inclusive digital transformation and regional development policies.

The project combines data preprocessing, exploratory data analysis, and interactive dashboard storytelling to transform national statistics into actionable intelligence.

- Analyze internet access trends from 2020 to 2024.

- Explore relationships between internet access and HDI.

# 🔍 Key Findings

- Internet penetration increased from **76.43% to 87.30% (2020–2024)**.
- Provincial disparity remains substantial, ranging from **32% to 96%**.
- The urban–rural gap persists at approximately **14.7 percentage points**.
- Correlation with HDI shows a declining trend, suggesting diminishing marginal impact.
- DKI Jakarta and Papua Pegunungan represent the widest observed digital access gap.
